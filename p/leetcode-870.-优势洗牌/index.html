Leetcode 870. 优势洗牌

Oct 09, 2022

链接： https://leetcode.cn/problems/advantage-shuffle/
难度： 中等

题目描述

给定两个大小相等的数组 nums1 和 nums2，nums1 相对于 nums2 的优势可以用满足 nums1[i] > nums2[i] 的索引 i 的数目来描述。

返回 nums1 的任意排列，使其相对于 nums2 的优势最大化。

示例 1：

输入： nums1 = [2,7,11,15], nums2 = [1,10,4,11]
输出： [2,11,7,15]

示例 2：

输入： nums1 = [12,24,8,32], nums2 = [13,25,32,11]
输出： [24,32,8,12]

提示：
  • 1 <= nums1.length <= 105
  • nums2.length == nums1.length
  • 0 <= nums1[i], nums2[i] <= 109

解题思路

贪心

为使得优势最大，需尝试从 nums1 中找出比 nums2 中较小的那部分数字更大，可以采用田忌赛马策略。

首先将两个数组排序，分别从头开始遍历（不妨设下标指针分别 i 和 j）：

  • 当 nums1[i] > nums2[j]，即找到了一个对应的优势点；
  • 否则，nums[i] 不会比剩余的 nums2 中的数字大了，可将其依次倒序放置到 nums2 的后半部分。

由于最后的结果要和 nums2 的初始排序对应，故可以按照 nums2 中的排序下标，通过下标找到对应元素。

func advantageCount(nums1 []int, nums2 []int) []int {
	n := len(nums1)
	idx2 := make([]int, 0, n)
	for i := 0; i < n; i++ {
		idx2 = append(idx2, i)
	}
	sort.Ints(nums1)
	sort.Slice(idx2, func(i, j int) bool {
		return nums2[idx2[i]] < nums2[idx2[j]]
	})

	ans := make([]int, n)
	j, k := 0, n-1
	for _, num1 := range nums1 {
		if num1 <= nums2[idx2[j]] {
			ans[idx2[k]] = num1
			k--
			continue
		}
		ans[idx2[j]] = num1
		j++
	}
	return ans
}
复杂度
  • 时间复杂度：$ O(N), N = len(nums1) = len(nums2) $
  • 空间复杂度：$ O(N) $
