Leetcode 172. 阶乘后的零

Mar 25, 2022

链接： https://leetcode-cn.com/problems/factorial-trailing-zeroes/
难度： 中等

题目描述

给定一个整数 n ，返回 n! 结果中尾随零的数量。

提示 n! = n * (n - 1) * (n - 2) * ... * 3 * 2 * 1

示例 1：

输入： n = 3
输出： 0
解释： 3! = 6 ，不含尾随 0

示例 2：

输入： n = 5
输出： 1
解释： 5! = 120 ，有一个尾随 0

示例 3：

输入： n = 0
输出： 0

提示：
  • 0 <= n <= 104

进阶： 你可以设计并实现对数时间复杂度的算法来解决此问题吗？

解题思路

数学

唯一可能在结尾出现0的情况是 2 * 5，所以需要统计所有阶乘因数中因子 2 和 5 的数量。

由于阶乘中 2 的数量一定是比 5 的数量多的，只需要统计 5 的数量，即为结尾零的数量。

func trailingZeroes(n int) int {
	ans := 0
	for i := 5; i <= n; i += 5 {
		t := i
		for t%5 == 0 {
			ans++
			t /= 5
		}
	}
	return ans
}
复杂度
  • 时间复杂度：$ O(n) $
  • 空间复杂度：$ O(1) $

优化

对于阶乘因数中含有因子 5 的因数数量，通过简单的除法运算即可求得 n / 5。

单部分因数中可能含有多个因子 5 ，还需要统计这一部分。例如含有因子 25 会包含两个因子 5，125 含有3个，同理进行统计，只记录多出来的这部分即可。

func trailingZeroes(n int) int {
	ans := 0
	for i := 5; i <= n; i *= 5 {
		ans += n / i
	}
	return ans
}
复杂度
  • 时间复杂度：$ O(\log{n}) $
  • 空间复杂度：$ O(1) $
