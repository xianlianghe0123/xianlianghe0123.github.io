Leetcode 565. 数组嵌套

Jul 17, 2022

链接： https://leetcode.cn/problems/array-nesting/
难度： 简单

题目描述

索引从 0 开始长度为 N 的数组 A，包含 0 到 N - 1 的所有整数。找到最大的集合 S 并返回其大小，其中 S[i] = {A[i], A[A[i]], A[A[A[i]]], ... } 且遵守以下的规则。

假设选择索引为 i 的元素 A[i] 为 S 的第一个元素，S 的下一个元素应该是 A[A[i]]，之后是 A[A[A[i]]]... 以此类推，不断添加直到 S 出现重复的元素。

示例：

输入： A = [5,4,0,3,1,6,2]
输出： 4
解释：
A[0] = 5, A[1] = 4, A[2] = 0, A[3] = 3, A[4] = 1, A[5] = 6, A[6] = 2.

其中一种最长的 S[K]:
S[0] = {A[0], A[5], A[6], A[2]} = {5, 6, 2, 0}

提示：
  • N 是 [1, 20,000] 之间的整数。
  • A 中不含有重复的元素。
  • A 中的元素大小在 [0, N-1] 之间。

解题思路

并查集

根据题意，由于数组中的元素不会重复，则整个数组中元素会构成一个或多个环。

枚举所有的初始元素，不断寻找直到出现环为止。

将已经查找过的元素做标记，降低时间复杂度。

func arrayNesting(nums []int) int {
	maxCnt := 0
	for i := range nums {
		cnt := 0
		for j := i; nums[j] != -1; {
			nums[j], j = -1, nums[j] // 标记已查找，并继续查找
			cnt++
		}
		if maxCnt < cnt {
			maxCnt = cnt
		}
	}
	return maxCnt
}
复杂度
  • 时间复杂度：$ O(N), N = len(nums) $
  • 空间复杂度：$ O(1) $
