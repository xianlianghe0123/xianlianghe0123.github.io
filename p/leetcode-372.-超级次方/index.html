Leetcode 372. 超级次方

Dec 05, 2021

链接： https://leetcode-cn.com/problems/super-pow/
难度： 中等

题目描述

你的任务是计算 ab 对 1337 取模，a 是一个正整数，b 是一个非常大的正整数且会以数组形式给出。ß

示例 1：

输入： a = 2, b = [3]
输出： 8

示例 2：

输入： a = 2, b = [1,0]
输出： 1024

示例 3：

输入： a = 1, b = [4,3,3,8,5,2]
输出： 1

示例 4：

输入： a = 2147483647, b = [2,0,0]
输出： 1198

提示：
  • 1 <= a <= 231 - 1
  • 1 <= b.length <= 2000
  • 0 <= b[i] <= 9
  • b 不含前导 0

解题思路

快速幂

两个因数相乘取模的值，与每个因数各自取模后相乘再取模的值一致。

显而易见是一道快速幂的题目，但是指数是以十进制数组的方式给出的，故每次步进时，是 a, a10, a100, …。同时，每一位上的数字可能的取值是 1~9，故用二进制快速幂求得每一位的计算结果和步进。

func superPow(a int, b []int) int {
	const MOD = 1337
	var fastPower = func(x, n int) int {
		ans := 1
		for ; n > 0; n >>= 1 {
			if n&1 != 0 {
				ans = ans * x % MOD
			}
			x = x * x % MOD
		}
		return ans
	}
	ans := 1
	a %= MOD
	for i := len(b) - 1; i >= 0; i-- {
		ans = ans * fastPower(a, b[i]) % MOD
		a = fastPower(a, 10)
	}
	return ans
}
复杂度
  • 时间复杂度：$ O(N), N = len(b) $
  • 空间复杂度：$ O(1) $
