Leetcode 6. Z 字形变换

Mar 01, 2022

链接： https://leetcode-cn.com/problems/zigzag-conversion/
难度： 中等

题目描述

将一个给定字符串 s 根据给定的行数 numRows ，以从上往下、从左到右进行 Z 字形排列。

比如输入字符串为 "PAYPALISHIRING" 行数为 3 时，排列如下：

P   A   H   N
A P L S I I G
Y   I   R

之后，你的输出需要从左往右逐行读取，产生出一个新的字符串，比如："PAHNAPLSIIGYIR"。

请你实现这个将字符串进行指定行数变换的函数：

string convert(string s, int numRows);

示例 1：

输入： s = “PAYPALISHIRING”, numRows = 3
输出： “PAHNAPLSIIGYIR”

示例 2：

输入： s = “PAYPALISHIRING”, numRows = 4
输出： “PINALSIGYAHRPI”
解释：

P     I    N
A   L S  I G
Y A   H R
P     I

示例 3：

输入： s = “A”, numRows = 1
输出： “A”

示例 4：

输入: s = ""
输出: 0

提示：
  • 1 <= s.length <= 1000
  • s 由英文字母（小写和大写）、',' 和 '.' 组成
  • 1 <= numRows <= 1000

解题思路

模拟

根据题设含义，模拟构造 Z 字形排列。由于并不关心每个字符所在的列，确定每个字符的行之后，追加到末尾即可。

注意： 对行为 1 的情况进行特判，直接返回原字符串。

func convert(s string, numRows int) string {
	if numRows == 1 {
		return s
	}
	dir, row := -1, 0
	matrix := make([][]byte, numRows)
	for i := range s {
		matrix[row] = append(matrix[row], s[i])
		if row == 0 || row == numRows-1 {
			dir = -dir
		}
		row += dir
	}
	return string(bytes.Join(matrix, nil))
}
复杂度
  • 时间复杂度：$ O(N), N = len(s) $
  • 空间复杂度：$ O(N) $

规律

对于每一行而言，可以发觉从左往右其在原字符串中的下标是有规律的，从而推导出下一个字符的坐标。假设第 row 行首个坐标为 i。

  • 对于第一行和最后一行，下一个坐标是 i+(numRows-1)*2;
  • 对于其它行，之后每连续两个字符的坐标分别是 i+(row-1)*2 和 i+(numRows-1)*2;

逐行推导下一个字符的位置，直接构造新字符串。

注意： 对行为 1 的情况进行特判，直接返回原字符串。

func convert(s string, numRows int) string {
	if numRows == 1 {
		return s
	}
	ans := make([]byte, 0, len(s))
	for row := 0; row < numRows; row++ {
		if row >= len(s) {
			break
		}
		ans = append(ans, s[row])
		for idx := row; idx < len(s); idx += (numRows - 1) << 1 {
			// 非首尾行，中间多加一次字符
			if row > 0 && row < numRows-1 && idx+(numRows-row-1)<<1 < len(s) {
				ans = append(ans, s[idx+(numRows-row-1)<<1])
			}
			if idx+(numRows-1)<<1 < len(s) {
				ans = append(ans, s[idx+(numRows-1)<<1])
			}
		}
	}
	return string(ans)
}
复杂度
  • 时间复杂度：$ O(N), N = len(s) $
  • 空间复杂度：$ O(1) $
