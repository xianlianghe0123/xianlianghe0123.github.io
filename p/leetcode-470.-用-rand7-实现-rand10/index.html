Leetcode 470. 用 Rand7() 实现 Rand10()

Sep 05, 2021

链接： https://leetcode-cn.com/problems/implement-rand10-using-rand7/
难度： 中等

题目描述

已有方法 rand7 可生成 1 到 7 范围内的均匀随机整数，试写一个方法 rand10 生成 1 到 10 范围内的均匀随机整数。

不要使用系统的 Math.random() 方法。

示例 1：

输入: 1
输出: [7]

示例 2：

输入: 2
输出: [8,4]

示例 3：

输入: 3
输出: [8,1,10]

提示：
  • rand7 已定义。
  • 传入参数: n 表示 rand10 的调用次数。
进阶：
  • rand7() 调用次数的 期望值 是多少 ?
  • 你能否尽量少调用 rand7() ?

解题思路

rand7 函数等概率的生成 1~7 的随机数，为生成 1~10的随机数，需要等概率的生成 10 的倍数个随机数。

将两次调用 rand7 生成的随机数相乘，将等概率的生成 49 个随机数，将最后 9 个数排除，则前 40 个数字的生成是等概率的，通过简单的模运算即可得到 1~10 的随机数。

每次生成最后的九个随机数时，重新进行上述步骤，直到生成前 40 个随机数为止。

期望：

容易求得前 i 次未生成的概率为$(\frac{9}{49})^i$，

则第 i 次生成的概率为$\frac{40}{49}(\frac{9}{49})^{i-1}$，调用 rand7 的次数为$2i\frac{40}{49}(\frac{9}{49})^{i-1}$

所以调用期望为:

$$ E = \sum_{i=1}^{+\infty}2i(\frac{9}{49})^{i-1}\frac{40}{49} = 2*\frac{40}{49}\sum_{i=1}^{+\infty}i(\frac{9}{49})^{i-1} $$

令 $S_n = \sum\limits_{i=1}^{n}i(\frac{9}{49})^{i-1}$，$\frac{9}{49}S_{n-1} = \sum\limits_{i=1}^{n}i(\frac{9}{49})^{i}$

$$ S_n - \frac{9}{49}S_{n-1} = \frac{40}{49}S_{n-1} = \sum\limits_{i=1}^{n}(\frac{9}{49})^{i-1} - n(\frac{9}{49}^n) = \frac{1-(\frac{9}{49})^n}{1-\frac{9}{49}} - n(\frac{9}{49})^n $$

故 $S_n = \frac{\frac{1-(\frac{9}{49})^n}{1-\frac{9}{49}} - n(\frac{9}{49})^n}{\frac{40}{49}}$

$$ E = 2*\frac{40}{49}\lim_{n\to+\infty}S_n = 2\lim_{n\to+\infty}\frac{1-(\frac{9}{49})^n}{1-\frac{9}{49}} - n(\frac{9}{49})^n = 2.45 $$

可以生成的期望为：2

func rand10() int {
	for { 
		pos := 7*(rand7()-1) + rand7()
		if pos <= 40 {
			return pos%10 + 1
		}
	}
}
复杂度
  • 时间复杂度：$ O(1) $
  • 空间复杂度：$ O(1) $
