Leetcode 396. 旋转函数

Apr 22, 2022

链接： https://leetcode-cn.com/problems/rotate-function/
难度： 中等

题目描述

给定一个长度为 n 的整数数组 nums 。

假设 arrk 是数组 nums 顺时针旋转 k 个位置后的数组，我们定义 nums 的 旋转函数 F 为：

  • F(k) = 0 * arrk[0] + 1 * arrk[1] + … + (n - 1) * arrk[n - 1]

返回 F(0), F(1), ..., F(n-1) 中的最大值 。

生成的测试用例让答案符合 32 位 整数。

示例 1：

输入： nums = [4,3,2,6]
输出： 26
解释：
F(0) = (0 * 4) + (1 * 3) + (2 * 2) + (3 * 6) = 0 + 3 + 4 + 18 = 25
F(1) = (0 * 6) + (1 * 4) + (2 * 3) + (3 * 2) = 0 + 4 + 6 + 6 = 16
F(2) = (0 * 2) + (1 * 6) + (2 * 4) + (3 * 3) = 0 + 6 + 8 + 9 = 23
F(3) = (0 * 3) + (1 * 2) + (2 * 6) + (3 * 4) = 0 + 2 + 12 + 12 = 26
所以 F(0), F(1), F(2), F(3) 中的最大值是 F(3) = 26 。

示例 2：

输入： nums = [100]
输出： 0

提示：
  • n == nums.length
  • 1 <= n <= 105
  • -100 <= nums[i] <= 100

解题思路

模拟

不进行任何旋转时，旋转函数值为 F(0) = 0 * nums[0] + 1 * nums[1] + ... + (n-2) * nums[n-2] + (n-1) * nums[n-1]
进行 1 次旋转后，旋转函数值为 F(1) = 1 * nums[0] + 2 * nums[1] + ... + (n-1) * nums[n-2] + 0 * nums[n-1]

二者求差，可得 F(1) - F(0) = nums[0] + nums[1] + ... + nums[n-2] - (n-1) * nums[n-1] = nums[0] + nums[1] + ... + nums[n-1] - n * nums[n-1]；

设 sum 为数组元素和，则F(1) = F(0) + sum - n * nums[n-1]

进行 2 次旋转，可以理解为在进行一次旋转生成新数组的基础上，再进行 1 次旋转。只是此次末尾元素变为了 nums[n-2]，故 F(2) = F(1) + sum - n * nums[n-2]

从而得到递推公式：F(k) = F(k-1) + sum - n * nums[n-k]

依据递推公式，求得所有旋转函数值，并从中找出最大值。

func maxRotateFunction(nums []int) int {
	sum := 0
	f := 0
	for i, num := range nums {
		sum += num
		f += i * num
	}
	maxF := f
	for i := len(nums) - 1; i > 0; i-- {
		f += sum - nums[i]*len(nums)
		if maxF < f {
			maxF = f
		}
	}
	return maxF
}
复杂度
  • 时间复杂度：$ O(N), N = len(nums) $
  • 空间复杂度：$ O(1) $
