剑指 Offer II 069. 山峰数组的顶部

Oct 14, 2021

链接： https://leetcode-cn.com/problems/B1IidL/
难度： 简单

题目描述

符合下列属性的数组 arr 称为 山峰数组（山脉数组） ：

  • arr.length >= 3
  • 存在 i（0 < i < arr.length - 1）使得：
    • arr[0] < arr[1] < ... arr[i-1] < arr[i]
    • arr[i] > arr[i+1] > ... > arr[arr.length - 1]

给定由整数组成的山峰数组 arr ，返回任何满足 arr[0] < arr[1] < ... arr[i - 1] < arr[i] > arr[i + 1] > ... > arr[arr.length - 1] 的下标 i ，即山峰顶部。

示例 1：

输入： arr = [0,1,0]
输出： 1

示例 2：

输入： arr = [1,3,5,4,2]
输出： 2

示例 3：

输入： arr = [0,10,5,2]
输出： 1

示例 4：

输入： arr = [3,4,5,1]
输出： 2

示例 5：

输入： arr = [24,69,100,99,79,78,67,36,26,19]
输出： 2

提示
  • 3 <= arr.length <= 104
  • 0 <= arr[i] <= 106
  • 题目数据保证 arr 是一个山脉数组

解题思路

暴力求解

遍历整个数组，找到最大值，返回其下标。

func peakIndexInMountainArray(arr []int) int {
	idx := 0
	for i, num := range arr {
		if arr[idx] < num {
			idx = i
		}
	}
	return idx
}
复杂度
  • 时间复杂度：$ O(N), N = len(arr) $
  • 空间复杂度：$ O(1) $

二分查找

每次获取区间中点 mid，将其与后一个比较：

  • 若 mid 大于 后一个元素，则在左半区间查找；
  • 否则，在右半区间查找。
func peakIndexInMountainArray(arr []int) int {
	l, r := 0, len(arr)
    for l != r {
        mid := (l+r)/2
        if arr[mid] > arr[mid+1] {
            r = mid
        } else {
            l = mid + 1
        }
    }
    return l
}
复杂度
  • 时间复杂度：$ O(\log{N}), N = len(arr) $
  • 空间复杂度：$ O(1) $
