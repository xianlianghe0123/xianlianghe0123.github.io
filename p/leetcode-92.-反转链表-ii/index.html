Leetcode 92. 反转链表 II

Oct 12, 2021

链接： https://leetcode-cn.com/problems/reverse-linked-list-ii/
难度： 中等

题目描述

给你单链表的头指针 head 和两个整数 left 和 right ，其中 left <= right 。请你反转从位置 left 到位置 right 的链表节点，返回 反转后的链表 。

示例 1：

输入： head = [1,2,3,4,5], left = 2, right = 4
输出： [1,4,3,2,5]

示例 2：

输入： head = [5], left = 1, right = 1
输出： [5]

提示：
  • 链表中节点数目为 n
  • 1 <= n <= 500
  • -500 <= Node.val <= 500
  • 1 <= left <= right <= n

进阶： 你可以使用一趟扫描完成反转吗？

解题思路

就地反转

使用虚拟的头结点，遍历找到待反转区域的上一个结点 prev，反转该区域，将 prev.Next 指向反转后的头部， 和尾部区域，

func reverseBetween(head *ListNode, left int, right int) *ListNode {
    newHead := &ListNode{
        Next: head,
    }
    // 跳过头
    prev := newHead
    for i := 1; i < left; i++ {
        prev = prev.Next
    }
    // 反转中间
    tail := (*ListNode)(nil)
    node := prev.Next
    for i := left; i <= right; i++ {
        node.Next, node, tail = tail, node.Next, node
    }
    // 连接头尾
    prev.Next.Next = node 
    prev.Next = tail
    return newHead.Next
}
复杂度
  • 时间复杂度：$ O(right) $
  • 空间复杂度：$ O(1) $
