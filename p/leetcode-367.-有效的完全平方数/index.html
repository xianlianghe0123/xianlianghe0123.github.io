Leetcode 367. 有效的完全平方数

Nov 04, 2021

链接： https://leetcode-cn.com/problems/valid-perfect-square/
难度： 简单

题目描述

给定一个 正整数 num ，编写一个函数，如果 num 是一个完全平方数，则返回 true ，否则返回 false 。

进阶： 不要 使用任何内置的库函数，如 sqrt 。

示例 1：

输入： num = 16
输出： true

示例 2：

输入： num = 14
输出： false

提示：
  • 1 <= num <= 231 - 1

解题思路

暴力求解

从 i = 1 开始每次累加 1，直到 i * i >= num 时为止。

若此时 i * i = num ，则返回 true；否则，返回 false 。

func isPerfectSquare(num int) bool {
	i := 1
	for ; i*i < num; i++ {
	}
	if i*i == num {
		return true
	}
	return false
}
复杂度
  • 时间复杂度：$ O(\sqrt{num}) $
  • 空间复杂度：$ O(1) $

二分查找

若 num 是一个完全平方数，则其平方根位于区间 [1, num] 中，可以使用二分查找求解，设区间中间为 mid，左右端点为 l 和 r ：

  • 当 mid * mid = num，返回 true
  • 当 mid * mid < num，在区间 [mid+1, r] 中查找；
  • 当 mid * mid > num，在区间 [l, mid-1] 中查找；

直到区间为空时，说明 num 不是完全平方数，返回 false。

func isPerfectSquare(num int) bool {
	for l, r := 1, num; l <= r; {
		mid := (l + r) / 2
		if mid*mid == num {
			return true
		} else if mid*mid < num {
			l = mid + 1
		} else {
			r = mid - 1
		}
	}
	return false
}
复杂度
  • 时间复杂度：$ O(\log{num}) $
  • 空间复杂度：$ O(1) $
