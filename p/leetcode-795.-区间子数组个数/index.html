Leetcode 795. 区间子数组个数

Nov 24, 2022

链接： https://leetcode.cn/problems/number-of-subarrays-with-bounded-maximum/
难度： 中等

题目描述

给你一个整数数组 nums 和两个整数：left 及 right 。找出 nums 中连续、非空且其中最大元素在范围 [left, right] 内的子数组，并返回满足条件的子数组的个数。

生成的测试用例保证结果符合 32-bit 整数范围。

示例 1：

输入： nums = [2,1,4,3], left = 2, right = 3
输出： 3
解释： 满足条件的三个子数组：[2], [2, 1], [3]

示例 2：

输入： nums = [2,9,2,5,6], left = 2, right = 8
输出： 7

提示：
  • 1 <= nums.length <= 105
  • 0 <= nums[i] <= 109
  • 0 <= left <= right <= 109

解题思路

滑动窗口

满足条件的数组，其所有取值范围必须为 ≤ right，并且其中至少有一个元素 ≥ left。

对于结尾为 i 的连续子数组，向前寻找的首个在区间 [left, right] 中的元 curMatchIdx，则从此元素开始，继续向前所有的 ≤ right 的元素均符合满足条件的子数组的起始位置。记在 curMatchIdx 之前的首个 > right 的元素为 start，则以 i 结尾的子数组个数为 curMatchIdx - start。

遍历整个数组，维护 start 和 curMatchIdx：

  • 每当遇到 > right 的元素时，更新 start 和 curMatchIdx 为当前下标；
  • 当遇到在 [left, right] 的元素时，仅更新 curMatchIdx 为当前下标。
func numSubarrayBoundedMax(nums []int, left int, right int) int {
	ans := 0
	curMatchIdx := -1
	start := -1
	for i, num := range nums {
		if num > right {
			start = i
			curMatchIdx = start
			continue
		}
		if left <= num && num <= right {
			curMatchIdx = i
		}
		ans += curMatchIdx - start
	}
	return ans
}
复杂度
  • 时间复杂度：$ O(N), N = len(nums) $
  • 空间复杂度：$ O(1) $
