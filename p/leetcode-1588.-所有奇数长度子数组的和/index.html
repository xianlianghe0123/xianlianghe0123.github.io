Leetcode 1588. 所有奇数长度子数组的和

Aug 29, 2021

链接： https://leetcode-cn.com/problems/sum-of-all-odd-length-subarrays/
难度： 简单

题目描述

给你一个正整数数组 arr ，请你计算所有可能的奇数长度子数组的和。

子数组 定义为原数组中的一个连续子序列。

请你返回 arr 中 所有奇数长度子数组的和 。

示例 1：

输入： arr = [1,4,2,5,3]
输出： 58
解释： 所有奇数长度子数组和它们的和为：
[1] = 1
[4] = 4
[2] = 2
[5] = 5
[3] = 3
[1,4,2] = 7
[4,2,5] = 11
[2,5,3] = 10
[1,4,2,5,3] = 15
我们将所有值求和得到 1 + 4 + 2 + 5 + 3 + 7 + 11 + 10 + 15 = 58

示例 2：

输入： arr = [1,2]
输出： 3
解释： 总共只有 2 个长度为奇数的子数组，[1] 和 [2]。它们的和为 3 。

示例 3：

输入： arr = [10,11,12]
输出： 66

提示：
  • 1 <= arr.length <= 100
  • 1 <= arr[i] <= 1000

解题思路

暴力求解

枚举出所有奇数子数组的 [i, j]，求得子数组的和，累加进结果中。

func sumOddLengthSubarrays(arr []int) int {
	sum := 0
	for i := 0; i < len(arr); i++ {
		for j := i; j < len(arr); j += 2 {
			t := 0
			for k := i; k <= j; k++ {
				t += arr[k]
			}
			sum += t
		}
	}
	return sum
}
复杂度
  • 时间复杂度：$ O(N^3), N = len(arr) $
  • 空间复杂度：$ O(1) $

前缀和

依次枚举出所有子数组的开始下标 i，对于每个 i ，遍历其后元素，求得子数组的和，每当子数组长度为奇数时，将该数组的和累加进结果中。

func sumOddLengthSubarrays(arr []int) int {
	sum := 0
	for i := 0; i < len(arr); i++ {
		t := 0
		for j := i; j < len(arr); j++ {
			t += arr[j]
			if (j-i)&1 == 0 {
				sum += t
			}
		}
	}
	return sum
}
复杂度
  • 时间复杂度：$ O(N^2), N = len(arr) $
  • 空间复杂度：$ O(1) $
